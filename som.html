Kohonen Self-Organising Maps

2017-18

Description

Implement an interactive and modern web-application for clustering different input alphabets of various handwritings in real-time using Kohonen’s Artificial Neural Network trained on the EMNIST database

Goals

Goal to introduce and explain Machine Learning to users of various ages and level of expertise. Inspired by Google’s QuickDraw, Andrej Karpathy’s ConvNetJS and general pattern recognition projects.

Challenges
  • Optimisation of algorithms when working with large quantities of data to improve processing speeds.
  • Data Visualisation on the front-end using D3.Js to create topological maps of Neural Network's output data
  • Sorting data according to labels.
  • Connecting front and back ends using Python's Flask web-framework.
Implementation
  • Thesis: 191 pages, 18000 words
  • Front-End: Bootstrap, JavaScript, CSS, HTML
  • Back-End: Python, NumPy, Pandas, Matplotlib, ArgsParse
  • Linking: Flask
